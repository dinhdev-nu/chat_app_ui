# Chat App UI

UI for real-time chat project with authenticated users, using Next.js 15 and TypeScript.

## 🚀 Features

- User Login/Register
- Real-time Chat UI
- Manage chats in rooms

## 🛠️ Settings

git clone https://github.com/dinhdev-nu/chat_app_ui.git
cd chat_app_ui
pnpm install
